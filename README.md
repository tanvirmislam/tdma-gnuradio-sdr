# TDMA MAC implementation on GNURadio with a IEEE 802.15.4 PHY Layer

The system includes:<br/>
        - IEEE 802.15.4 PHY Layer<br/>
        - TDMA MAC Layer<br/>
        - Ecryption of payloads using AES-CBC encryption<br/>
